+) 문제 해결을 위한 추가 조사

1. 문자 배열 조사하기
 -> 문자열 저장
 -> 문자열 -> 문자 (처리)

char arr[16]; //크기가 16인 문자 배열
-> 15개의 문자 저장
-> 끝에 NULL 문자(\0)가 있어야 함
 1) 초기화 -> 자동 추가
 2) 수동 -> 직접 명시적으로 추가

char str[] = "hi";  -> 자동 추가

char str[3] = {'h', 'i', '\0'};  -> 수동 추가

초기화 기준
1) {} -> 문장 or 문자 하나가 여러 개
2) '' -> 문자 하나

----------------------------------------------------------------------

2. 1차원 문자 배열 NULL 문자로 초기화하기

char str[16] = {0}; // 모든 요소가 '\0'로 초기화됨
char str[16] = {};
